# B4勉強会
この勉強会では，信号処理の基礎を再確認するとともに，より実践的な技術を勉強します．
教材を参考に，ソースコードを作り実行してください．
勉強した内容はパワーポイントで説明してもらいます．

## 勉強会の実施形態
各自，教材を参考に進めてください．  
月に１回ミーチィングを行い，そこで勉強した内容をパワーポイントで発表してもらいます．
他人が見てわかりやすいよう，資料を作成してください．
